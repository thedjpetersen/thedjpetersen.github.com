Proxying Grunt

Like my last blog post I have been digging more into client side project managers. I have recently switched from using Yeoman to just using Grunt. If in some way you disagree with some of Yeoman's opinions this allows you to build a more compatible workflow. I personally like it because it allows me to set up my project the way I please.

I found a nicer way to proxy requests using the request npm module from mikeal. In the following I add it to my middleware. It is the most durable solution I have found yet.

request = require('request')

connect:
  livereload:
    options:
      port: 9001
      middleware: (connect, options) ->
        [
          lrSnippet,
          folderMount(connect, "build"),
          (req, res, next) ->
            if req.url.indexOf('/api/') == 0
              req
                .pipe(request('http://localhost:4567' + req.url.slice(4)))
                .pipe(res)
        ]

That code will proxy all requests on the /api url root to localhost:4567 .
